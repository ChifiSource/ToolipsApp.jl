Toolips Markdown

A simple markdown to Toolips Component parser. Wraps markdown components into a Toolips.divier

Module Composition

  • ToolipsMarkdown
ToolipsMarkdown.tmd — Function

Toolips Markdown

tmd(name::String = "tmd", s::String = "") -> ::Component

Turns a markdown string into a Toolips Component. Markdown will always use default styling.

example

route("/") do c::Connection
    mymdcomp = tmd("mainmarkdown", "# Hello! [click](http://toolips.app/)")
    write!(c, mymdcomp)
end
